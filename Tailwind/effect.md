# Box Shadow
设置元素的 shadow 效果
* shadow-none
* shadow-sm
* shadow
* shadow-md
* other...
* shadow-inner

# Box Shadow Color
* shadow-white
* shadow-slate-50
* other...

# Opacity
控制元素的 opacity
* opacity-0
* opacity-5 -> opacity: 0.05
* other...
* opacity-100 -> opacity: 1

# Blur
apply blur effect to element
* blur-none -> filter: blur(0)
* blur-sm -> filter:blur(4px)
* blur
* other...

# Brightness
apply brightness effect to element
* brightness-0
* brightness-50
* other...
